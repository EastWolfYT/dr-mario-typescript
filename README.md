# Dr Mario TypeScript Game

A browser game inspired by Dr Mario, built with TypeScript and Webpack.  
The project recreates core gameplay mechanics such as falling capsules, rotation, collision detection, virus generation, match checking, scoring and game-over logic.

## Features

- Browser-based gameplay
- TypeScript codebase
- Webpack module bundling
- Falling pill/capsule logic
- Pill rotation
- Collision detection
- Virus generation
- Match detection
- Score system
- Level system
- Game-over state
- Keyboard controls
- Local high score storage
- Modular project structure

## Tech Stack

- TypeScript
- JavaScript
- Webpack
- HTML
- CSS
- Browser APIs
- LocalStorage
